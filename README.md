# Day 19 - Sum and Average of Array

##  Description
This Java program accepts array elements from the user, calculates the sum of all elements, and finds the average of the array.

##  Features
- Takes array size from the user
- Accepts array elements using Scanner
- Calculates the sum of all elements
- Calculates the average of the array
- Displays the results

##  Technologies Used
- Java
- Scanner Class
- Arrays
- For Loop

##  Sample Input
Enter the number of elements: 5

Enter the array elements:
1
2
3
4
5

##  Sample Output
Sum = 15
Average = 3.0

##  Concepts Practiced
- User Input
- Arrays
- Looping
- Sum Calculation
- Average Calculation
- Type Casting (double)
